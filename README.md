# Python School Projects

## Overview
Collection of Python exercises and small projects completed during coursework at Minot State University. These practice projects cover fundamental programming concepts including functions, file handling, data structures, and basic data processing.

## What You'll Find Here
- **Practice exercises** from Python programming courses
- **Small utility scripts** and command-line tools
- **Basic data analysis** and automation tasks
- **Code samples** demonstrating core Python skills

## Skills Practiced
- Variables, loops, conditionals
- Functions and user input
- File I/O and string processing
- Lists, dictionaries, basic algorithms
- Error handling and debugging

## How to Run
1. Clone repository
2. Open terminal/command prompt
3. Navigate to any project folder
4. Import needed file if running the notebook
5. Run: python filename.ipynb

**Note:** These are educational exercises - perfect for reviewing Python fundamentals and coursework completion.
